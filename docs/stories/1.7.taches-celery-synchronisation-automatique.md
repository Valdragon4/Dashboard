# Story 1.7: Tâches Celery pour Synchronisation Automatique

## Status

Done

## Story

**As a** utilisateur,
**I want** que mes comptes soient synchronisés automatiquement une fois par jour,
**so that** mes données sont toujours à jour sans intervention manuelle.

## Acceptance Criteria

1. Tâche Celery `sync_all_bank_accounts()` créée dans `finance/tasks.py` qui synchronise tous les comptes avec `auto_sync_enabled=True`
2. Tâche Celery `sync_bank_account(account_id)` pour synchronisation d'un compte spécifique
3. Configuration Celery Beat pour exécuter `sync_all_bank_accounts()` quotidiennement (configurable via variable d'environnement)
4. Gestion des erreurs dans les tâches avec retry automatique (utilisant les mécanismes Celery)
5. Notification à l'utilisateur en cas d'échec répété (via Django messages ou email) - implémentation basique avec logging pour l'instant
6. Logging structuré des synchronisations dans `SyncLog` (déjà géré par SyncService)
7. Tests des tâches Celery avec mocks du SyncService

```python
            minute=int(os.getenv('BANK_SYNC_SCHEDULE_MINUTE', 0))
        ),
    },
}
```

**Tâche Celery avec retry** :

```python
from celery import shared_task
from celery.exceptions import Retry
import logging

logger = logging.getLogger(__name__)

@shared_task(
    autoretry_for=(ConnectionTimeoutError, RateLimitError),
    retry_kwargs={'max_retries': 3, 'countdown': 60},
    retry_backoff=True,
    retry_backoff_max=600,
)
def sync_bank_account(account_id: int, sync_type: str = "automatic"):
    """Synchronise un compte bancaire spécifique."""
    try:
        account = Account.objects.get(id=account_id)
        result = SyncService.sync_account(account, sync_type=sync_type)
        return result
    except Account.DoesNotExist:
        logger.error(f"Compte {account_id} introuvable")
        raise
    except Exception as e:
        logger.error(f"Erreur lors de la synchronisation du compte {account_id}: {str(e)}")
        raise
```

**Gestion des erreurs dans `sync_all_bank_accounts()`** :

```python
@shared_task
def sync_all_bank_accounts():
    """Synchronise tous les comptes avec auto_sync_enabled=True."""
    accounts = Account.objects.filter(
        auto_sync_enabled=True,
        bank_connection__isnull=False
    )
    
    results = {
        "total": accounts.count(),
        "success": 0,
        "errors": 0,
        "errors_details": []
    }
    
    for account in accounts:
        try:
            result = SyncService.sync_account(account, sync_type="automatic")
            if result["success"]:
                results["success"] += 1
            else:
                results["errors"] += 1
                results["errors_details"].append({
                    "account_id": account.id,
                    "error": result.get("error", "Unknown error")
                })
        except Exception as e:
            results["errors"] += 1
            results["errors_details"].append({
                "account_id": account.id,
                "error": str(e)
            })
            logger.error(f"Erreur lors de la synchronisation du compte {account.id}: {str(e)}")
    
    logger.info(f"Synchronisation terminée: {results['success']} succès, {results['errors']} erreurs")
    return results
```
