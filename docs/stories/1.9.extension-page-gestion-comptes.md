# Story 1.9: Extension de la Page de Gestion des Comptes

## Status

Done

## Story

**As a** utilisateur,
**I want** voir le statut de synchronisation de mes comptes sur la page de gestion des comptes,
**so that** je peux rapidement identifier les comptes synchronisés automatiquement.

## Acceptance Criteria

1. Page `/accounts/` étendue pour afficher le statut de synchronisation :
   - Nouvelle colonne "Synchronisation" dans le tableau des comptes
   - Affichage uniquement pour les comptes avec `bank_connection` associée
   - Les comptes sans connexion bancaire s'affichent normalement sans erreur
2. Badge/indicateur visuel pour les comptes avec synchronisation automatique activée :
   - Badge "Auto-sync" ou icône pour les comptes avec `auto_sync_enabled=True`
   - Badge "Manuel" ou absence de badge pour les comptes sans synchronisation automatique
   - Design cohérent avec les autres badges de la page
3. Affichage de la date/heure de la dernière synchronisation :
   - Utiliser `BankConnection.last_sync_at` si disponible
   - Format lisible (ex: "Il y a 2 heures", "Aujourd'hui à 14:30", "Hier à 10:15", ou date complète si > 7 jours)
   - Afficher "Jamais" si aucune synchronisation réussie
4. Affichage du statut (succès, échec, en cours) avec code couleur :
   - Badge de statut avec code couleur :
     - Vert (success) : Dernière synchronisation réussie
     - Rouge (error) : Dernière synchronisation échouée
     - Jaune (syncing) : Synchronisation en cours
     - Gris (pending) : En attente de synchronisation
   - Utiliser `BankConnection.sync_status` pour déterminer le statut
   - Afficher le statut de manière claire et visible
5. Bouton de synchronisation manuelle directement depuis la liste des comptes :
   - Bouton "Sync" ou icône pour chaque compte avec `bank_connection`
   - Appel asynchrone de la tâche Celery `sync_bank_account()` via endpoint API
   - Feedback visuel immédiat (bouton désactivé pendant la synchronisation, spinner, etc.)
   - Message de confirmation/erreur via notification ou toast
6. Lien vers la page de gestion des connexions pour les comptes connectés :
   - Lien "Gérer" ou icône vers `/bank-connections/` pour les comptes avec `bank_connection`
   - Optionnellement, lien direct vers la connexion spécifique si possible
   - Design cohérent avec les autres liens de la page

## Integration Verification

### Technical Implementation Details

**Optimisation des requêtes DB** :

```python
@login_required
def accounts(request: HttpRequest) -> HttpResponse:
    """Affiche la liste des comptes avec statut de synchronisation."""
    # Optimiser les requêtes avec select_related et prefetch_related
    accounts = Account.objects.filter(owner=request.user).select_related(
        "bank_connection"
    ).prefetch_related(
        "bank_connection__sync_logs"
    )
    
    # Pour chaque compte, calculer les informations de synchronisation
    for account in accounts:
        if account.bank_connection:
            # Récupérer le dernier SyncLog réussi depuis le prefetch
            last_success_log = None
            for log in account.bank_connection.sync_logs.all():
                if log.status == SyncLog.Status.SUCCESS:
                    if not last_success_log or log.completed_at > last_success_log.completed_at:
                        last_success_log = log
            
            account.sync_info = {
                "status": account.bank_connection.sync_status,
                "last_sync_at": account.bank_connection.last_sync_at,
                "auto_sync_enabled": account.auto_sync_enabled,
                "last_success_log": last_success_log,
            }
        else:
            account.sync_info = None
    
    return render(request, "finance/accounts.html", {"accounts": accounts})
```

**Endpoint API pour synchronisation** :

```python
@login_required
def account_sync_api(request: HttpRequest, account_id: int) -> JsonResponse:
    """API endpoint pour synchroniser manuellement un compte."""
    try:
        account = Account.objects.select_related("bank_connection").get(
            id=account_id, owner=request.user
        )
    except Account.DoesNotExist:
        return JsonResponse({"success": False, "error": "Compte introuvable."}, status=404)
    
    if not account.bank_connection:
        return JsonResponse(
            {"success": False, "error": "Ce compte n'a pas de connexion bancaire."},
            status=400
        )
    
    from finance.tasks import sync_bank_account
    
    try:
        task_result = sync_bank_account.delay(account.id, sync_type=SyncLog.SyncType.MANUAL)
        return JsonResponse({
            "success": True,
            "message": "Synchronisation démarrée.",
            "task_id": str(task_result.id) if hasattr(task_result, "id") else None,
        })
    except Exception as e:
        return JsonResponse(
            {"success": False, "error": f"Erreur lors du démarrage de la synchronisation : {str(e)}"},
            status=500
        )
```

**JavaScript pour synchronisation** :

```javascript
function syncAccount(accountId) {
  const button = document.querySelector(`[data-account-id="${accountId}"] .sync-button`);
  const originalText = button.textContent;
  
  // Désactiver le bouton et afficher spinner
  button.disabled = true;
  button.innerHTML = '<span class="spinner">⟳</span> Synchronisation...';
  
  // Récupérer le token CSRF
  const csrftoken = document.querySelector('[name=csrfmiddlewaretoken]').value;
  
  // Appel API
  fetch(`/api/accounts/${accountId}/sync/`, {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'X-CSRFToken': csrftoken,
    },
  })
  .then(response => response.json())
  .then(data => {
    if (data.success) {
      // Afficher message de succès
      showNotification('Synchronisation démarrée avec succès', 'success');
      // Optionnellement, mettre à jour le statut affiché
    } else {
      // Afficher message d'erreur
      showNotification('Erreur : ' + data.error, 'error');
    }
  })
  .catch(error => {
    console.error('Erreur:', error);
    showNotification('Erreur de connexion au serveur', 'error');
  })
  .finally(() => {
    // Réactiver le bouton
    button.disabled = false;
    button.textContent = originalText;
  });
}
```
