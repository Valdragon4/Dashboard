📚 Documentation et commandes alternatives

Import par ligne de commande

Boursobank, Hello bank, etc.

Si vous préférez utiliser la ligne de commande :

python manage.py import_bank_statement <username> /chemin/releve.csv "Compte Boursobank" --profile boursobank

Profils disponibles :

  • generic - Format générique
  • boursobank - BoursoBank
  • hellobank - Hello bank (compte courant)
  • hellobank_livret - Hello bank Livret (ajuste automatiquement le solde initial)

Le compte est créé automatiquement s'il n'existe pas.

Note pour les livrets : Le profil hellobank_livret extrait le solde actuel depuis la première ligne du CSV et ajoute automatiquement une transaction d'ajustement si nécessaire.
